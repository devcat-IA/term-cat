# 🐱 term-cat

Uma versão simples e colorida do comando `cat`, com opção de numerar linhas.

## 💻 Uso

```bash
./term-cat [-n] arquivo

    -n: numera as linhas

🧪 Exemplo

./term-cat -n exemplo.txt

🔧 Compilação

make

📁 Estrutura

    src/: código fonte

    include/: headers

    Makefile: compilação
